# Vikings Draft Site
# Overview
This website is a data-driven NFL draft tool used to analyze the Minnesota Vikings draft process in 2026. In this website, you'll see a  combination of player evaluation metrics, exclusive team needs, along with customizable models aimed to simulate NFL draft decision making and evaluate player fits for the Vikings.

# Features
- Interactive Draft Board
- Player data and rankings
- Team needs analysis

This project was built to explore how computer and data science can be applied to real-world sports decision-making. Overall, this personal passion project reflects my interest in combining computer science, analytics, and football to better understand how NFL teams evaluate talent.

# Technology
- Python
- Streamlit
- Pandas
